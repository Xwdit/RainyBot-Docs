# 类: NewFriendRequestEvent  
[(返回目录)](README.md)  
  
**继承自:** [RequestEvent](RequestEvent.md)  
  
RainyBot的添加好友请求事件类，其实例记录了与一次添加好友请求事件相关的数据  
  
## 枚举  
  
enum **RespondType**  
  
可用于回应请求的回应类型  
  
- **ACCEPT** = 0  
接受好友添加请求  
  
- **REFUSE** = 1  
拒绝好友添加请求  
  
- **REFUSE_BLACKLIST** = 2  
拒绝好友添加请求并加入黑名单
