# 类: BotOfflineEvent  
[(返回目录)](README.md)  
  
**继承自:** [BotEvent](BotEvent.md)  
  
RainyBot的机器人账号离线事件类，其实例记录了一次机器人账号离线事件的相关数据  
  
## 枚举  
  
enum **ReasonType**  
  
机器人账号的离线原因类型的枚举  
  
- **ACTIVE** = 0  
机器人账号主动离线  
  
- **FORCE** = 1  
机器人账号被挤下线(如其他客户端登录等情况)  
  
- **DROPPED** = 2  
机器人账号意外掉线(网络原因或服务器原因等)
