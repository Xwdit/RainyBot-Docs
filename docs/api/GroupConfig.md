## 描述  
  
这是RainyBot的群组配置类，通常代表一个对应实例，储存了与群组各类配置有关的信息   
对此类实例中内容的更改不会直接影响群组的配置，需要在更改完成后于群组实例中将此类实例设定为群组配置
