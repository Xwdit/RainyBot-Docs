# 类: NudgeEvent  
[(返回目录)](README.md)  
  
**继承自:** [ActionEvent](ActionEvent.md)  
  
RainyBot的戳一戳事件类，记录了聊天中的一个戳一戳事件的相关数据  
  
## 枚举  
  
enum **SubjectType**  
  
戳一戳事件发生的上下文位置类型，可以是好友聊天或群聊  
  
- **FRIEND** = 0  
事件发生在好友聊天中  
  
- **GROUP** = 1  
事件发生在群组聊天中
